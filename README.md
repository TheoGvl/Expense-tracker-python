# Mini Expense Tracker 

Μια σύγχρονη και πρακτική εφαρμογή καταγραφής καθημερινών εξόδων. Ο χρήστης μπορεί να προσθέτει αγορές και η εφαρμογή υπολογίζει δυναμικά το συνολικό ποσό σε πραγματικό χρόνο. Το project δημιουργήθηκε με Python και Flet, με έμφαση στη δυναμική δημιουργία UI (Dynamic UI) και την ασφάλεια των δεδομένων εισόδου Input Validation.

## Τι ακριβώς κάνει;
* **Δυναμική Λίστα:** Κάθε φορά που ο χρήστης προσθέτει ένα νέο έξοδο, δημιουργείται αυτόματα μια νέα εγγραφή  στην οθόνη χωρίς να χρειάζεται ανανέωση όλης της σελίδας.
* **Αυτόματος Υπολογισμός:** Το συνολικό ποσό σε: € ανανεώνεται αστραπιαία με κάθε νέα προσθήκη.
* **Προστασία από Σφάλματα (Error Handling):** Διαθέτει σύστημα ελέγχου που αποτρέπει την εφαρμογή από το να «κρασάρει» αν ο χρήστης ξεχάσει να συμπληρώσει ένα πεδίο ή αν γράψει γράμματα αντί για αριθμούς στο πεδίο του ποσού.


**Βασικές προγραμματιστικές τεχνικές που εφαρμόστηκαν:**
1. **Dynamic UI Generation:** Χρήση της εντολής `controls.append()` για την προσθήκη νέων γραφικών στοιχείων (Containers, Rows, Texts) εν ώρα εκτέλεσης (runtime).
2. **Διαχείριση Εξαιρέσεων (`try...except`):** Έλεγχος των δεδομένων εισόδου (π.χ. μετατροπή string σε float) για την αποφυγή `ValueError` και την ομαλή εμπειρία χρήστη (UX).
3. **Έξυπνη Διάταξη (Layout):** Χρήση της ιδιότητας `expand=True` για την αυτόματη προσαρμογή του πλάτους των πεδίων, και `ScrollMode.AUTO` για την υποστήριξη scrolling όταν οι εγγραφές γίνουν πολλές.
4. **Μορφοποίηση Αριθμών (String Formatting):** Χρήση f-strings με την ιδιότητα `:.2f` για τον σωστό περιορισμό των δεκαδικών ψηφίων σε νομισματική μορφή.
